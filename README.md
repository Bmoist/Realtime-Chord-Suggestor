# Realtime-Chord-Suggestor
Imagination always helps in improvising, especially when the chords are not fixed. Can we have machines inform us on a realtime basis?

My vision for this project is a simple GUI (working with a predetermined bpm) and performs chord identification per beat. It then feeds the history data to a Markov chain/HMM/neural network/etc. to predict the next chord to be played.
